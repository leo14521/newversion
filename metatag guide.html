메타태그 직접 설치

메타 태그란 추천 위젯 노출 시 아이템명과 아이템 이미지, 가격 정보 등이 제대로 표시되기 위해 필요한 태그입니다.
아래 메타 태그를 아이템 상세 페이지에 삽입해주세요. 메타 태그는 </head> 코드 직전에 삽입하시는 것을 권장합니다.

1 주의사항

1. 아래 메타 태그는 대부분 Open Graph protocol 에 명시된 표준으로서, 추가하실 경우 SNS 및 검색엔진 최적화에 도움이 됩니다.
2. content 속성에 값을 넣으실 때, 특수 문자의 경우 적절하게 html encoding을 수행하셔야 합니다.
